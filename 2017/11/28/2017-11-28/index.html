某年某月某大宝

初识

2017年11月12的一个晚上，我朋友给我说给你介绍一个女朋友吧！给我了她的第一张照片，看到照片的第一眼我就感觉这就是我的幸运女神。
*注：我是一个注重眼缘的人，看到的第一眼喜欢就是喜欢！

初见

经过一周的接触，每天我们总有聊不完的话题，从日常到小说，从游戏到电影，每天总能对她了解的多一点。
终于2017年11月18日的上午，我们第一次见面。第一眼看到他的时候我感觉这就是我想要的，不需要美若天仙，不需要肥瘦合适，我看到的只有她。
美丽的外表，优美的声音和清澈的大眼睛。所以我决定追求她，为了自己的幸福！

告白

2017年11月24日我选择了告白，或许在别人眼里我们才相互接触2周，在一起的时间不超过24个小时。匆匆忙忙的告白是一件很莽撞的事情，会让她感觉我是一个轻浮的人，没有认真的对待这份感情。但是在我眼里24个小时已经足够让我踏入万丈深渊，在见到她的第二天起，我已经没有办法正常工作和生活。一周的时间干了n件在以前自己看来就很傻的事情，所以我要告白。及时不成功我也曾经尝试过，与其慢慢坠入爱河我选择了轰轰烈烈。

相恋

或许收到幸运女神的眷顾，25日的晚上我告白成功了。我们在一起了！虽然你们看来这份爱情来得很容易，但是在我看来更应该值得珍惜，毕竟我们相爱了！

完

这就是我的女神–>
